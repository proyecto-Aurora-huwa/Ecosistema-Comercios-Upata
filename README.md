# Ecosistema de Comercios Upata - Alianza Aurora

### Introducción
Este proyecto, iniciado el **3 de marzo de 2026**, funciona bajo el umbral de **Proyecto Aurora**. Nuestra meta es crear una red de comercios en la ciudad de Upata que permita a los ciudadanos comprar desde la comodidad de sus hogares a través de esta plataforma y recibir sus productos mediante un servicio de delivery directo.

### Objetivos Actuales
* **Ventas en Bolívares:** La plataforma está diseñada para que los negocios locales vendan sus productos y cobren en bolívares venezolanos, facilitando el uso diario para todos.
* **Comodidad del Usuario:** Permitir que los pobladores chequeen disponibilidad, precios y hagan sus pedidos online de manera rápida.
* **Apoyo a la Comunidad:** Crear un grupo de comercios que genere beneficios y datos útiles para la comunidad local Con objetivos de expansión.

### Visión a Futuro (Web3)
Aunque hoy operamos de forma tradicional, este ecosistema sienta las bases sólidas para la tecnología del futuro. Nuestra intención es:
* **Integrar Web3:** En una fase posterior, cuando la moneda **Huwa** sea respaldada por una fundación, ya tendremos aquí el ecosistema apropiado y la red de comercios lista para que la moneda empiece a circular de forma segura y descentralizada.
